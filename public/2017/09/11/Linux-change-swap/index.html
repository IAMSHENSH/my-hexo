Linux 调整 Swap

发表于 2017-09-11 | 分类于 Linux

关于在 Linux 安装后调整 Swap 的方法

调整步骤

  1. 切换到超级用户

  2. 在 「/root/」目录下创建一个 2GB 名为「myswaofile」的交换文件

  3. 设置文件权限，以保护此 swap 分区

  4. 使用 mkswap命令，让此文件设置 「swap」

  5. 启动交换文件

  6. 为了确保开机启动，在「/etc/fstab」末尾加入：

    CoreOS 无此文件

  7. 查看系统中交换文件设置情况

  8. 使用 free 确认再次确认

其他命令

打开所有交换空间

关闭所有交换空间
